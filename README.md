# Assignment-1
Cross-Market-Analysis
Overview:
An interactive data analytics dashboard built with Python, MySQL, and Streamlit that performs cross-market analysis across Cryptocurrency, Oil Prices, and Stock Market data using SQL queries and dynamic visualizations.

Features:

🏠 Market Overview — Date-filtered averages for Bitcoin, Oil, S&P 500, and NIFTY with a daily market snapshot table
💰 Cryptocurrencies — Top coins by market cap, ATH analysis, supply metrics
📈 Crypto Prices — Daily price trends, highest/lowest prices, % change analysis
🛢️ Oil Prices — Yearly averages, COVID crash analysis, volatility per year
📊 Stock Prices — Ticker-based analysis for NASDAQ, S&P 500, and NIFTY
🔗 Join Queries — 10 cross-market SQL JOIN queries with charts
🌐 Cross Market Analysis — Top 3 coin selector with date filter, trend charts, and comparison
